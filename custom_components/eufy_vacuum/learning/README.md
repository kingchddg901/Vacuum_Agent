# Eufy Vacuum Learning System

## What it does
Tracks cleaning runs and learns:
- room duration
- battery usage
- drift
- cleaning patterns

## What it stores
- completed job JSON
- learned job stats
- learned room stats
- optional CSV exports

## What it powers
- smart duration estimates
- room progress inference
- queue tracking confidence

## Optional system
This system is completely optional and does not affect:
- vacuum operation
- job execution
- core integration behavior
